This website displays my analysis on Presidential Approval Ratings vs Crime Rate over the last 60 years while practicing/showing off some of my HTML, CSS and Bootstrap skills. On this webpage you will find 3 different tabs that you can explore: Plots, Comparison,and Data.

  • Plots: This tab will let you choose which crime type (Total Crime, Violent, Property, Murder) you would like to see against the presendential approval rating with graphs and a description
  • Comparison: On this page you will see all the scatter graphs next to each other
  • Data: Here you'll see where the data was pulled from and the table with all combined data that was used in the analysis
